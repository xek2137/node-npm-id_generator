## Description

    Npm package for generate random ID numbers. Contains upper / lower-case letters and numbers. 
    It's up to you how long it should be.

## Installation

Use yarn or npm manager:

```
npm add node-npm-id_generator
yarn add node-npm-id_generator
```

## How to use:

```js
const {randomID} = require('node-npm-id_generator');

console.log(randomID(8)); // 32SGxsA9
```
